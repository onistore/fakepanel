Apa Itu VCraftSMP?

VCraftSMP Adalah Server Minecraft Survial-RPG Yang Memiliki Banyak Fitur Menarik, Seperti Custom Enchant, Ranks, DLL. Di Server Ini, Kamu Bisa Claim Area Untuk Membuat Rumah. Server Ini Akan Terus Update! 

Players Yang Friendly

Di Sini, Playernya Di Jamin Friendly & Baik. Jadi, Kamu Bisa Bermain Disini Bersama-Sama! 

Selalu Update

Server Ini Selalu Update, Mulai Dari Fitur Maupun Version!

Fast Respond

Disini, Adminnya Fast Respond, Kamu Bisa Melaporkan Bug/Player Di Discord Server Kami! 

Cara Join Ke Server

Ini Step-Step Yang Harus Kamu Lakukan Untuk Join Ke Server.

1

Tambah Server

Pergi Ke Multiplayer, Lalu Tekan "Add Server"
2

Masukkan IP Server

Masukkan play.vcraftsmp.my.id Di Bagian Server Address, Lalu Pencet "Done".

Dan Selesai!
